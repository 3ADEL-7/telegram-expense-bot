# Telegram Expense Bot

بوت تيليجرام لتسجيل المصاريف مع تقارير يومية وأسبوعية وشهرية.

## المميزات

- تسجيل المصروف مع وصف اختياري وتصنيف عبر `#تصنيف`
- عداد مستقل لكل مستخدم
- تعديل المصروف وحذفه
- تصفير عداد المصاريف لكل مستخدم مع إعادة ترقيم السجل الحالي
- ميزانية شهرية مع متابعة المتبقي أو التجاوز
- ملخص أسبوعي تلقائي
- تقرير تصنيفات للشهر الحالي
- تخزين محلي داخل `JSON`
- لا يحتاج مكتبات خارجية، فقط `Node.js 18+`

## الإعداد

1. افتح الملف `.env`
2. ضع توكن البوت في `BOT_TOKEN`
3. شغّل:

```powershell
npm.cmd start
```

## الأوامر

- `/start`
- `/help`
- `/add 25 قهوة`
- `/add 35 أوبر #مواصلات`
- `/today`
- `/week`
- `/month`
- `/recent`
- `/categories`
- `/delete 3`
- `/edit 3 45 أوبر #مواصلات`
- `/budget 2000`
- `/budget off`
- `/autosummary on`
- `/autosummary off`
- `/resetcounter`

## أمثلة

```text
18 لاتيه #قهوة
45 أوبر #مواصلات
/edit 2 60 مطعم #أكل
```

## تشغيل 24/7 على Railway

1. ارفع المشروع إلى GitHub
2. في Railway اختر `Deploy from GitHub Repo`
3. أضف المتغيرات:

```text
BOT_TOKEN=توكن_البوت
BOT_TIMEZONE=Asia/Riyadh
```

4. أضف `Volume`
5. اربطه على:

```text
/app/data
```

6. تأكد أن أمر التشغيل:

```text
npm start
```
